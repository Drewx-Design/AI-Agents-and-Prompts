# Quick Health Check

Regular system health verification - use every few hours.

```
QUICK SYSTEM HEALTH CHECK

Verify 4-file system status:

FILE CONSISTENCY:
- All cross-references valid?
- No conflicting information?
- Current Focus accurate?

VELOCITY CHECK:
- Tasks completing within estimates?
- Circuit breakers triggering appropriately?
- Performance baseline being met?

QUALITY STATUS:
- Code meeting RULES.md standards?
- Working Systems intact?
- No regression issues?

RED FLAGS:
- Any anti-patterns emerging?
- Technical debt accumulating?
- Integration issues brewing?

Provide brief health summary and any recommended actions.
```
